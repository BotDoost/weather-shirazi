نمونه کار هواشناسی با جاوا اسکریپت

طراحی شده توسط شیرازی

توضیحات پروژه

این یک نمونه کار پیشرفته هواشناسی است که با استفاده از جاوا اسکریپت طراحی شده است. این برنامه قادر است اطلاعات آب و هوایی را از یک API دریافت کرده و به صورت زیبا و تعاملی نمایش دهد.

ویژگی‌های این پروژه شامل نمایش دما، وضعیت جوی، سرعت باد، رطوبت و فشار هوا می‌باشد. همچنین امکان جستجوی شهرهای مختلف و مشاهده اطلاعات هواشناسی آنها وجود دارد.

این پروژه به صورت کاملاً ریسپانسیو طراحی شده و در تمام دستگاه‌ها به خوبی نمایش داده می‌شود. از تکنیک‌های مدرن CSS مانند Flexbox، Grid و انیمیشن‌های جذاب استفاده شده است.
